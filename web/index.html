CaseDash

A compact dashboard for dedicated PC telemetry screens: small USB/HDMI panels, case-mounted displays, or a secondary screen beside the main monitor.

Download GitHub
  1. Install the MSI.
  2. Run CaseDash.
  3. Right-click, choose a layout and theme.
  4. Configure the display and enable startup when ready.

Download

Install the current release

The recommended download is the x64 MSI installer. Standalone executable, ZIP, and checksum artifacts are available on the same release page.

Latest release All releases Source repository

The MSI installs CaseDash.exe into C:\Program Files\CaseDash, can launch it after installation, and leaves startup setup to the app menu.

Features

Built to sit on a panel and stay useful

Glanceable telemetry

CPU, GPU, memory, network, storage, board sensors, and time in a stable native dashboard.

Live layout editing

Constraint-based layouts with guides for cards, widgets, spacing, and text anchors.

Theme system

Every app theme is available here, derived from the same base colors used by CaseDash.

Small native app

A focused Windows executable with fast rendering and no generic monitoring-suite sprawl.

Small-screen layouts

Shipped layouts cover 5-inch 800x480 panels, portrait displays, and wide 1920x480 screens.

Display setup

Configure Display fits the dashboard, remembers placement, and prepares a matching wallpaper.

Hardware

Supported telemetry

GraphicsAMD Radeon through ADLX. NVIDIA through NVML, with presented-FPS capture through ETW.
MotherboardsGigabyte SIV and MSI Center SDK temperature and fan telemetry.
WindowsCPU, memory, network, storage, clock, display placement, and startup behavior.
